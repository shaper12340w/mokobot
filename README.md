<p align="center">
🎵 **MokoBot은 강력한 음악 기능을 제공하는 Discord 봇입니다.**  
서버 관리, 게임, 출석 체크, 운세 기능 등 다양한 편의 기능이 추가될 예정이며,  
AI LLM 모델을 활용한 기능도 계획 중입니다.
</p>
